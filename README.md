# Capstone Project

**Buzy Moms Recipe App**
========================================================================================================================================

**Description:**

Buzy Moms Recipe app is a full stack application inspired by hardworking mothers who have to juggle a career and kids.

**Tech used**

* Chrome Web Browser
* React JS
* MongoDB
* Github
